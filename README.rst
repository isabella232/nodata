Usage
-----

Seamfilling
~~~~~~~~~~~

::

    nodata seamfill [OPTIONS] SRC_PATH DST_PATH

    Options:
    -m, --max-search-distance INTEGER  Maximum filling radius [default = 4]
    -n, --nibblemask                   Nibble filled nodata areas.
    --help                             Show this message and exit.
